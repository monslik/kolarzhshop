# Инструкция по настройке Telegram уведомлений

## Шаг 1: Создание бота в Telegram

1. Откройте Telegram и найдите бота **@BotFather**
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и придумайте имя для вашего бота
4. После создания бот выдаст вам **токен** (выглядит как `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)
5. **Скопируйте этот токен** - он понадобится позже

## Шаг 2: Получение Chat ID

Есть несколько способов узнать ваш Chat ID:

### Способ 1: Через @userinfobot
1. Найдите бота **@userinfobot** в Telegram
2. Начните с ним диалог (отправьте `/start`)
3. Бот покажет ваш Chat ID (число, например: `123456789`)

### Способ 2: Через ваш бот
1. Напишите вашему боту (тому, который вы создали) любое сообщение
2. Откройте в браузере: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
   (замените `<ВАШ_ТОКЕН>` на токен из шага 1)
3. Найдите в ответе поле `"chat":{"id":123456789}` - это и есть ваш Chat ID

## Шаг 3: Настройка config.js

1. Откройте файл `config.js` в редакторе
2. Замените `YOUR_BOT_TOKEN_HERE` на токен бота из шага 1
3. Замените `YOUR_CHAT_ID_HERE` на Chat ID из шага 2

Пример:
```javascript
const TELEGRAM_CONFIG = {
    BOT_TOKEN: '123456789:ABCdefGHIjklMNOpqrsTUVwxyz',
    CHAT_ID: '123456789'
};
```

## Шаг 4: Проверка работы

1. Оформите тестовый заказ на сайте
2. Проверьте, что сообщение пришло в Telegram

## Важные замечания

- **Не публикуйте** файл `config.js` в открытых репозиториях (добавьте его в `.gitignore`)
- Токен бота дает полный доступ к боту - храните его в секрете
- Chat ID может быть отрицательным числом для групп (начинается с `-`)

## Устранение проблем

### Сообщения не приходят
1. Проверьте, что токен бота правильный
2. Проверьте, что Chat ID правильный
3. Убедитесь, что вы написали боту хотя бы одно сообщение (если используете способ 2)
4. Откройте консоль браузера (F12) и проверьте ошибки

### Ошибка "chat not found"
- Убедитесь, что вы написали боту сообщение перед использованием
- Проверьте правильность Chat ID
